# DeepRAG-Coder：基于 LangChain Deep Agents + RAG 的智能编程助手

> 一个开箱即用、深度思考、真正理解代码仓库的 AI 编程助手

## ✨ 核心理念

DeepRAG-Coder 融合了 **LangChain Deep Agents 生态**与**代码仓库 RAG（检索增强生成）技术**，旨在打造一个真正理解项目全局、能够自主规划和执行复杂编码任务的智能编程助手。

Deep Agents 是 LangChain 团队推出的“开箱即用”智能体框架——一个 **batteries-included** 的 agent harness，具备任务规划、文件系统上下文管理、子智能体委派和长期记忆等内置能力。它构建在 LangChain 和 LangGraph 之上，既保留了 LangGraph 的流式传输、持久化、检查点等生产级能力，又提供了比基础 `create_agent` 更丰富的开箱即用功能。

在此基础上，我们引入**专为代码设计的 RAG 管道**——通过 AST 解析保障分块完整性、向量库实现语义检索、结合仓库知识图谱提供全局结构——让智能体能够真正“读懂”你的代码库，而不仅仅是关键词匹配。

## 🚀 核心特性

### 🤖 Deep Agents 智能体核心

Deep Agents 是一个**模型无关**的框架，支持任何支持工具调用的 LLM——前沿 API、开源权重模型或本地模型。其核心能力包括：

- **任务规划与委派**：采用协调员-工作者架构，主智能体规划任务并委派给专门的子智能体，每个子智能体在隔离的上下文中运行。内置通用子智能体，镜像主智能体的能力。
- **虚拟文件系统**：支持读写、编辑、搜索可插拔的本地/沙箱/远程后端，让智能体像人类开发者一样操作文件。
- **上下文管理**：内置上下文压缩机制，允许模型在适当时机压缩自身上下文窗口；同时保留完整对话历史在虚拟文件系统中，支持摘要后的上下文恢复。
- **持久化记忆**：可插拔的状态和存储后端，支持跨会话记忆召回。
- **人机协作**：支持在关键决策点暂停并等待人工审批。
- **Skills 技能系统**：可复用的行为模块，智能体可按需加载。
- **MCP 原生支持**：完全支持模型上下文协议（Model Context Protocol），可通过标准接口连接数据库、API、文件系统等。

**版本 0.6 核心新增能力**：

- **代码解释器（Code Interpreter）** ：为智能体提供一个可编程工作区，可转换数据、协调工具调用，并将中间结果保留在运行时状态中，无需每次往返模型。
- **Harness Profiles**：按模型调优的配置包，让 Kimi、Qwen、DeepSeek 等开源权重模型也能达到生产级性能，成本仅为闭源前沿 API 的 **1/20**。
- **DeltaChannel**：仅存储每个步骤的增量变化而非完整序列化值，使长运行智能体的检查点存储减少高达 **100 倍**。
- **ContextHubBackend**：与 LangSmith Context Hub 集成，版本化管理智能体的技能、策略和记忆。

**版本 0.5 核心能力**：

- **异步子智能体（Async Subagents）** ：智能体可启动非阻塞后台任务，用户可在子智能体并发工作的同时继续与主智能体交互。异步子智能体是有状态的，可在任务执行过程中接收后续指令或中途调整方向。

### 📚 RAG 代码仓库理解

代码与自然语言不同——结构严格，不能随意按字符分割。DeepRAG-Coder 的 RAG 管道专为代码设计：

- **AST 引导的语义分块**：基于抽象语法树（AST）解析保障代码分块的逻辑完整性，避免在函数或类中间截断。
- **语义向量检索**：将代码块转换为向量嵌入，实现“按含义搜索”而非仅靠关键词匹配。支持混合检索策略（BM25 关键词匹配 + 嵌入语义搜索）。
- **仓库知识图谱**：将代码仓库解析为“符号节点 + 调用关系边”的知识图谱，叠加 LLM 语义摘要与向量索引。让智能体能够查询符号、调用关系、文件结构、影响面和受影响测试。
- **上下文感知生成**：将用户问题、相关代码片段和项目全局结构融合为语义完整的 prompt，让大模型在完整的上下文中进行推理生成。
- **多语言支持**：支持 10+ 编程语言的代码解析与检索。

## 🔧 技术栈

| 层级           | 技术选型                                                                                 |
| -------------- | ---------------------------------------------------------------------------------------- |
| **智能体框架** | LangChain Deep Agents（基于 LangChain + LangGraph）                                      |
| **推理模型**   | 任意支持工具调用的 LLM：OpenAI、Anthropic、Google，或开源模型（Kimi、Qwen、DeepSeek 等） |
| **代码解析**   | Tree-sitter（AST 引导的代码分块与解析）                                                  |
| **向量数据库** | Qdrant / Milvus / Chroma（语义向量存储与检索）                                           |
| **知识图谱**   | Neo4j / Memgraph（符号节点 + 调用关系边）                                                |
| **部署与观测** | LangSmith（追踪、评估、监控、部署）                                                      |
| **MCP 协议**   | 原生支持 Model Context Protocol                                                          |

## 🚀 快速开始

### 安装 Deep Agents Code（dcode）

Deep Agents Code 是 LangChain 官方提供的终端编码智能体，基于 Deep Agents SDK 构建：

```bash
# 一键安装 dcode
curl -LsSf https://langch.in/dcode | bash
```

或通过 Python 包管理器安装：

```bash
uv add deepagents
```

### 初始化代码仓库 RAG 索引

```bash
# 初始化代码仓库的 RAG 索引
deeprag-coder init --repo ./my-project

# 或使用 igraph 构建代码知识图谱
npx igraph-cli build --repo ./my-project
```

### 启动智能体

```python
from deepagents import create_deep_agent

# 创建 Deep Agent
agent = create_deep_agent(
    model="openai:gpt-5.5",           # 或 anthropic:claude-sonnet-4-6
    tools=[rag_search, code_analyzer], # 自定义 RAG 工具
    system_prompt="你是一个代码仓库专家，能够检索和理解整个项目的代码。",
)

# 运行智能体
result = agent.invoke({
    "messages": [{
        "role": "user",
        "content": "项目中认证模块的 JWT 验证逻辑在哪里？帮我分析一下安全性。"
    }]
})
```

### 配置异步子智能体

```python
from deepagents import AsyncSubAgent, create_deep_agent

agent = create_deep_agent(
    model="openai:gpt-5.5",
    subagents=[
        AsyncSubAgent(
            name="code_analyzer",
            description="对代码仓库进行深度分析",
            url="https://my-agent-server.dev",
            graph_id="code_analysis_agent",
        ),
    ],
)
```

异步子智能体返回任务 ID 后立即执行，主智能体可继续与用户交互。

## 📖 使用场景

| 场景               | 描述                                           |
| ------------------ | ---------------------------------------------- |
| **代码问答**       | 用自然语言询问代码逻辑、API 用法、模块依赖关系 |
| **智能代码补全**   | 基于仓库上下文（不仅是当前文件）进行语义补全   |
| **跨文件重构**     | 理解全局依赖关系，安全地重构代码               |
| **代码审查**       | 基于项目历史修复实例和最佳实践进行智能评审     |
| **文档生成**       | 自动生成项目文档、API 文档和代码注释           |
| **Bug 定位与修复** | 通过语义检索快速定位问题代码，结合历史修复建议 |
| **新手上手**       | 为新加入项目的开发者提供代码库导航和解答       |
